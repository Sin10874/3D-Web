# 3D粒子交互系统

一个基于Three.js和MediaPipe的实时交互3D粒子系统，支持手势控制。

## ✨ 功能特性

- 🎨 **多种粒子模型**：爱心、花朵、土星、云朵、佛像、烟花、猫、三棱镜
- 👋 **手势控制**：通过摄像头检测双手张合，实时控制粒子群的缩放和扩散
- 🌈 **颜色自定义**：使用颜色选择器调整粒子颜色
- 🖥️ **全屏模式**：支持全屏显示
- 💫 **流畅动画**：粒子系统实时响应手势变化
- 🎯 **简洁UI**：现代化的玻璃拟态设计

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:3000`

### 构建生产版本

```bash
npm run build
```

## 🎮 使用说明

### 手势控制

- **张开双手**：粒子群扩散（距离越大，扩散越大）
- **合拢双手**：粒子群收缩（距离越小，收缩越紧）
- **单手或无手**：粒子恢复默认状态

### 控制面板

1. **选择模型**：点击按钮切换不同的粒子模型
2. **粒子颜色**：使用颜色选择器调整粒子颜色
3. **全屏模式**：点击按钮进入/退出全屏

## 🛠️ 技术栈

- **Three.js** - 3D图形渲染
- **MediaPipe Hands** - 手势识别
- **Vite** - 构建工具

## 📋 系统要求

- 支持WebGL的现代浏览器（Chrome, Firefox, Safari, Edge）
- 摄像头访问权限
- 建议使用Chrome浏览器以获得最佳性能

## 🎨 粒子模型说明

- **❤️ 爱心**：经典心形曲线
- **🌸 花朵**：五瓣花朵形状
- **🪐 土星**：带光环的球体
- **☁️ 云朵**：自然云朵形态
- **🧘 佛像**：冥想坐姿轮廓
- **🎆 烟花**：爆炸式球形扩散
- **🐱 猫**：猫咪轮廓（含耳朵和尾巴）
- **💎 三棱镜**：棱柱形状

## 📝 注意事项

- 首次运行需要授权摄像头访问
- 确保环境光线充足以获得更好的手势识别效果
- 双手需要完整出现在摄像头视野内
- 建议在距离摄像头0.5-1米的位置使用
